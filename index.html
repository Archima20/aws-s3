Qué es HTML?

HTML es el lenguaje de marcado estándar utilizado para crear la estructura y el contenido de una página web. Con HTML, se pueden definir diferentes elementos como encabezados, párrafos, imágenes, enlaces y listas, entre otros.

Con HTML, puedes definir la estructura de tu página utilizando etiquetas como "html", "head" y "body". También puedes agregar contenido como texto, imágenes, enlaces, listas y mucho más utilizando etiquetas como "p", "img" y "a".

HTML, abreviatura de Lenguaje de Marcado de Hipertexto (en inglés, Hypertext Markup Language), es el lenguaje central utilizado en la World Wide Web. Inicialmente diseñado para dar una estructura semántica a documentos científicos, se ha adaptado para describir la disposición básica de páginas web y aplicaciones en línea.

Por ejemplo, HTML se emplea para definir qué parte de un documento es un título, qué se considera una lista y qué constituye una imagen. También se utiliza para crear enlaces a otras páginas, insertar imágenes, aplicar formato en cursiva a fuentes y realizar muchas otras tareas.

¿Es HTML un lenguaje de programación?

La discusión sobre si HTML es un lenguaje de programación es continua. La mayoría lo considera un "lenguaje de marcado" en lugar de un lenguaje de programación, ya que HTML se enfoca en estructurar el contenido y no en ejecutar funciones. Se argumenta que HTML es un lenguaje declarativo, que se centra en indicar "qué" debe hacerse en lugar de "cómo" hacerlo. Aunque esta discusión es interesante, no tiene un impacto práctico en cómo se utiliza HTML.

Qué es CSS?

Cascading Style Sheets (Hojas de Estilo en Cascada). CSS se encarga de dar estilo y apariencia a una página web. Con CSS, puedes controlar la presentación visual de los elementos HTML. Por ejemplo, se puede definir el color del texto de un párrafo con p { color: blue; }.

Qué es JavaScript?

Es un lenguaje de programación que se utiliza para hacer que las páginas web sean interactivas y dinámicas. A través de JavaScript, se pueden realizar acciones como validar formularios, manipular elementos HTML, realizar cálculos y aplicar efectos visuales. A diferencia de HTML y CSS, es un lenguaje de programación completo con su propia sintaxis y reglas. Se puede escribir scripts directamente en tu página web o vincular archivos externos. ej: console.log("¡Hola, mundo!");.
